Frequently Asked Questions

What is the difference between var, let and const?
'var' is function-scoped and old, 'let' is block-scoped and re-assignable, and 'const' is block-scoped and not re-assignable. Use const by default, switch to let if value changes and avoid var.
What is the difference between map(), forEach() and filter()?
  • 1. forEach() → Loops through an array, lets you run code on each item, but returns undefined.
  • 2. map() → Loops through an array and returns a new array with transformed values.
  • 3. filter() → Loops through an array and returns a new array with items that pass a condition.
What are arrow functions and how are they different from regular functions?
Arrow functions (()=>{}) are a shorter way to write functions. Unlike regular functions, they don’t have their own this, arguments, super or new.target.
How does JavaScript Promises work?
A Promise is an object that represents the eventual completion (or failure) of an asynchronous operation. It can be in one of three states:
  • 1. Pending → still running
  • 2. Fulfilled → finished successfully (.then)
  • 3. Rejected → failed (.catch)
How does closure work in JavaScript?
A 'closure' is created when a function remembers and can access variables from its outer scope, even after that outer function has finished executing.
